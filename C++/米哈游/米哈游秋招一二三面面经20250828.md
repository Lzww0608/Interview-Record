## 判断当前机器是大端还是小端；
1. 联合体
```cpp
bool isLittleEndian() {
    union {
        int i;       // 4字节
        char c[4];   // 1字节数组
    } u;
    u.i = 0x12345678;  // 假设int为32位，十六进制表示为0x12 0x34 0x56 0x78
    
    // 低地址（c[0]）存储的是0x78则为小端，存储0x12则为大端
    return (u.c[0] == 0x78);
}
```
